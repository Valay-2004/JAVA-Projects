# Bank Account Simulator

A console-based banking application that simulates common banking operations.

## Key Features
- Create and manage bank accounts
- Support for multiple account types
- Deposit and withdraw functionality
- Balance inquiry
- Transaction logging
- Input validation and custom exceptions

## Learning Outcomes
- Practical application of OOP principles
- Handling state changes and persistence
- Separation of responsibilities across classes
- Working with monetary values using `BigDecimal`

## How to Run
1. Navigate to the directory: `cd Bank\ Account\ Simulator/BankAccountSimulator`
2. Compile: `javac *.java`
3. Run: `java Main`
