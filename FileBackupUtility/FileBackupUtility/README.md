# Student Grade Management System (StudentGMS)

A console-based student management system for maintaining academic records.
## Key Features
- Add, update, and remove student records
- Assign and manage grades or marks
- Calculate averages, percentages, or GPA
- Display student reports
- Search functionality by identifier
- Optional file-based data persistence

## Learning Outcomes
- Working with collections such as `ArrayList`
- Implementing search and basic sorting logic
- Structuring medium-sized console applications
- Managing data through simple storage mechanisms

## How to Run
1. Navigate to the directory: `cd StudentGMS`
2. Compile: `javac *.java`
3. Run: `java Main`
